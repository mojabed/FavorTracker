## Favor Tracker
Keeps track of which favor dailies have been completed in a given day through a simple checklist.

### Slash Commands
```
/favortracker show | hide | toggle | reset | lock | unlock
```

### Features
A simple window that provides a checklist of each favor, the npc related to the favor, and the location of the freerunner post board.

## Installation

1. Download the latest release
2. Copy `FavorTracker/` into:
   ```
   Documents/Elder Scrolls Online/live/AddOns/
   ```
3. Restart ESO or `/reloadui`
